# 统计各位数字都不同的数字个数

## 🎯 问题描述（来源于LeetCode）
```TEXT
给你一个整数 n ，统计并返回各位数字都不同的数字 x 的个数，其中 0 <= x < 10^n 。
```
## 💻 代码实现
```python
class Solution:
    def countNumbersWithUniqueDigits(self, n: int) -> int:
        if n == 0:
            return 1
        total = 1  
        for k in range(1, n+1):
            count = 9
            for i in range(9, 9 - (k-1), -1):
                count *= i
            total += count
        return total
```
