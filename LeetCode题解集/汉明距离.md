# 汉明距离
## 问题描述（来源于LeetCode）
两个整数之间的 [汉明距离](https://baike.baidu.com/item/%E6%B1%89%E6%98%8E%E8%B7%9D%E7%A6%BB) 指的是这两个数字对应二进制位不同的位置的数目。

给你两个整数 `x` 和 `y`，计算并返回它们之间的汉明距离。
## 代码实现
```python
class Solution:
    def hammingDistance(self, x: int, y: int) -> int:
         return bin(x^y).count("1")
```
## 复杂度分析
- 时间复杂度$O(Logn)$
- 空间复杂度$O(1)$
