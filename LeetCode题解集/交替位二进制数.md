# 交替位二进制数

## 🎯 问题描述（来源于LeetCode）
```TEXT
给定一个正整数，检查它的二进制表示是否总是 0、1 交替出现：换句话说，就是二进制表示中相邻两位的数字永不相同。
```
## 💻 代码实现

```python
class Solution:
    def hasAlternatingBits(self, n: int) -> bool:
        while n:
            a=n&1
            n>>=1
            b=n&1
            if a^b==0:
               return False
        return True
```
