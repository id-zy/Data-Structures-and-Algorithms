# 题目表述（来源LeetCode）
**斐波那契数** （通常用 `F(n)` 表示）形成的序列称为 **斐波那契数列** 。该数列由 `0` 和 `1` 开始，后面的每一项数字都是前面两项数字的和。也就是：
	F(0) = 0，F(1) = 1
	F(n) = F(n - 1) + F(n - 2)，其中 n > 1
给定 `n` ，请计算 `F(n)`
# 代码实现
```python
class Solution:
    def fib(self, n: int) -> int:
        a = [0,1] 
        i=2
        a[0]=0
        a[1]=1
        if n>1:
            for i in range(2,n+1):
                a.append(a[i-1]+a[i-2])
        return a[n]
```
# 复杂度分析
- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
