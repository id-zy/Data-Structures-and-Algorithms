# 七进制数
## 问题描述（来源于LeetCode）
给定一个整数 `num`，将其转化为 **7 进制**，并以字符串形式输出。
## 代码实现
```python
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num==0:
            return '0'
        pid = 0
        if num < 0:
            num = -num
            pid =1
        x= ''
        while num:
            a = num % 7
            num =num // 7
            x=str(a)+x
        if pid:
            return '-'+x
        return x
```
## 复杂度分析
- 时间复杂度$O(Log7N)$
- 空间复杂度$O(Log7N)$
